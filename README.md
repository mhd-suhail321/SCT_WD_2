# React Calculator

A clean, modern calculator built with React featuring smooth animations and keyboard support.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Keyboard input support
- Error handling for invalid expressions
- Responsive design
- Smooth button and display animations (Framer Motion)

## Technologies

- React
- Framer Motion
- CSS (custom styling)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/calculator-react.git

# Navigate to project directory
cd calculator-react

# Install dependencies
npm install

# Start development server
npm start
